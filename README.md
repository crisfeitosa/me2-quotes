
# Challenge ME2 Softwares

Aplicativo de citações desenvolvido com React Native, Typescript, Styled Components, Context Api, Expo, Axios,
React Navigation e Formik

## Instruções

Instalar as dependências 

```bash
npm install
```

Iniciar a aplicação 

```bash
npm start
```
